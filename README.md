Инструкция для пользователя (обновленная версия 2.0)
Описание проекта:
Программа представляет собой усовершенствованный графический интерфейс PowerShell Script, предназначенный для комфортного вырезания определённого числа строк из текстовых файлов и отправки их в буфер обмена. Теперь также доступна возможность отменить предыдущие изменения, выбирать кодировку файла и получать подробную информацию о ходе операций.

Как пользоваться программой?
Шаг 1. Запуск программы
Дважды кликните исполняемый скрипт add-type.ps1, чтобы открыть программу.

Шаг 2. Выбор файла
Нажмите кнопку «Выбрать файл». Появится окно проводника, где вам нужно выбрать текстовый файл формата .txt. Путь к файлу появится ниже кнопки.

Шаг 3. Установка параметров
Количество строк: Введите количество строк, которые хотите извлечь из файла и отправить в буфер обмена.
Выбор кодировки: Используйте выпадающее меню, чтобы выбрать подходящую кодировку вашего файла (например, UTF-8, ASCII, Unicode).
Шаг 4. Обработка файла
Нажмите кнопку «Обработать», чтобы начать обработку файла. Программа извлекает указанные строки, помещает их в буфер обмена и обновляет файл, сохраняя изменения.

Результат операции будет отображён в статусе внизу окна.

Шаг 5. Возможность отката
Если вы случайно ошиблись с количеством строк или выбрали неверную кодировку, воспользуйтесь функцией отмены изменений («Отменить операцию»). Это вернёт ваш файл в предыдущее состояние.

Шаг 6. Завершение работы
По окончании всех манипуляций нажмите кнопку «Закрыть», чтобы выйти из программы.

Важные моменты:
Если указанное количество строк превышает размер файла, программа уведомит вас об ошибке.
Приложение создаёт временные резервные копии ваших файлов, гарантируя сохранность данных при любых операциях.
Рекомендуется сохранять важную информацию заранее!
Приятной работы с программой!



Инструкция для пользователя ( для версии 1.0 )

Описание проекта:

Программа представляет собой графический интерфейс PowerShell Script, предназначенный для удобного вырезания первых заданных строк из выбранного текстового файла и помещения их в буфер обмена. Это полезно для быстрого редактирования файлов больших размеров вручную.

Как пользоваться программой?
Шаг 1: Запуск программы
Откройте программу двойным кликом по исполняемому скрипту add-type.ps1.

Шаг 2: Выбор файла
Нажмите кнопку «Выбрать файл».
Выберите нужный текстовый файл (.txt) из папки на вашем компьютере.
После выбора файла путь отображается в окне ниже кнопки.
Шаг 3: Укажите количество строк
Введите нужное число строк, которое хотите удалить из начала файла и поместить в буфер обмена.

Например, если введёте цифру 5, программа удалит первые пять строк текста и поместит их в буфер обмена Windows.

Шаг 4: Обработка файла
Нажмите кнопку «Продолжить», чтобы запустить процесс обработки файла. Программа автоматически выполнит операцию удаления указанных строк и отправит их в буфер обмена.

Результат операции появится в строке статуса.

Шаг 5: Завершение работы
По завершении всех операций нажмите кнопку «Закрыть», чтобы завершить работу программы.

Важные моменты:
Если введённое вами количество строк превышает размер файла, программа выдаст ошибку.
Все изменения сохраняются сразу же после нажатия кнопки «Продолжить». Обязательно сохраняйте резервную копию вашего файла перед обработкой!
Приятной работы с программой!
